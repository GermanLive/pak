Embedthis Pak™ Features

Pak has the following major features:

  • Install, list and uninstall packages into a local application
  • Automatically retrieves and installs dependant paks
  • Update and upgrade paks and their dependencies
  • Manage pak versions and dependencies
  • Support complex version expressions using Semantic Versioning 2.0
  • Manages a local cache of retrieved packages
  • Provides a global catalog to search for public paks
  • Integrated with the Bower and NPM catalogs for thousands of of extra paks
  • Supports GitHub and NPM as the primary pak storage repositories
  • Online Pak Catalog to locate installable paks
  • Publish and retract paks in the Pak Catalog
  • Post install configuration of paks by selectively exporting pak contents into local "lib" directory
  • Customization by package.json overrides for customize installation of third party packages
  • Open Source

Nice Parts

While Pak has a strong feature set, there are some characteristics that make Pak particularly sweet.

  • Local cache of paks supports fast offline use and local component sharing between apps
  • Can publish, modify and retract paks in the Pak catalog
  • Web portal to manage your published paks
  • Packages are described via a Package.json file that is compatible with Node
  • Fast operation
