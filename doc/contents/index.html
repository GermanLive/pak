Embedthis Pak™

Embedthis Pak is a package manager for application components. Pak makes it easy to retrieve, integrate and manage components for applications. Unlike other package managers, Pak will automatically install and configure downloaded packages — no more manual file installation each time you upgrade a library.

The Online Pak Catalog provides tested, integrated componens that are designed work together. Pak makes it easy to publish packs into the catalog and uses GitHub or NPM as the back-end storage for packages. Pak also supports overriding and wrapping existing third party packages to customize which files are installed to where.

Why Pak?

Existing package managers like NPM are excellent, but the published components are typically tuned for Node and not for general application or web components.

Pak calls packages "paks" — this is not a misspelling.
