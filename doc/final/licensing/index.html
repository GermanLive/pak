Licensing

Pak is licensed under two separate licenses. You may use either, provide you fully comply with the governing terms.

  • Pak Commerical License.
  • GNU GPL Open Source license.

Pak Commercial License

The Pak commercial license is a royalty-free license that provides you with the following benefits:

  • The right to develop and modify the software to embed in your products
  • The ability to redistribute your products with the software
  • The ability to keep your source code private
  • Simple royalty-free, one-time purchase options
  • Software maintenance for updates, upgrades and security patches
  • Priority software support services

If you require a commercial license or commercial support for Pak, please contact us at: licensing@embedthis.com.

Open Source License

Pak my be used with the GPL Open Source license. This is ideal for evaluating Pak, for internal use, stand-alone use, or for integrating exclusively with other GPL software. If you intend to distribute your software and link Pak with your application, the GPL requires you to provide public and full access to all source code and documentation on a nondiscriminatory basis. If you do not wish to provide source code to your application, you must purchase a commercial license.

Evaluating Pak

If you are evaluating Pak internally, and not distributing any products containing Pak, then you may be able to use the GPL license for this evaluation. Once complete, you may purchase a commercial license so that you can distribute your product. You may evaluate, develop, modify, and embed Pak in your application or product provided it is not distributed. You may copy and modify the source code provided all copies fall under the GPL license.

Stand Alone Use

If you are using Pak in a stand-alone manner as free-standing binaries, and are not integrating the Pak source code or libraries with your software to create a derivative work, then the GPL license may be sufficient. Stand-alone use does not permit linking Pak using either static or dynamic linking.

Use with Other GPL Software

If you are integrating Pak source or libraries into your application and your software is also fully licensed under the GPL license, you may use Pak with the GPL license and embed Pak in your applications or devices. You do not need any other agreement or license — the GPL is all you require.

Embedding with Non-GPL Software

If Pak is embedded into your application, or your application is linked using static or dynamic linking with libraries containing any Pak source code and you do not wish to publish your source code under the GPL, then you will need the Pak Commercial license.

Please contact licensing@embedthis.com if you have any questions about licensing Pak.
